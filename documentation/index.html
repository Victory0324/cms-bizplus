1) Introduction - top

BizPlus is a complete and full featured business, agency and company related content management system (CMS) that is made with Codeigniter PHP framework. This software or script is very easy to use by admin and end users. Admin can change almost everything in this web software.

2) Features - top

The important features of this script are:

  • Attractive, simple, easy interface
  • 100% responsive layout
  • Clean and fresh codes
  • Protected bad access for page
  • In-short statistics in the admin dashboard
  • Complete Multi Language System
  • Unlimited Customer Registration with Email Verification
  • Product and shop section with cart and checkout
  • Guest Checkout option included
  • Payment Method Integration (PayPal and Stripe)
  • Manage testimonials by admin
  • Manage team member by admin
  • Manage services by admin
  • Manage photo and video gallery by admin
  • Manage blog and categories by admin
  • Unlimited page creation and manage
  • Easy installation on server
  • Reset password option for customer
  • Reset password option for admin panel
  • Secured against SQL injection, XSS Attack etc.
  • And many more...

3) Requirements - top

Before starting to install our item, make sure you fulfill the following requirements:

  1. PHP Version >= 7.0
  2. MySQL Version >= 5.6
  3. GD Library Enabled
  4. mod_rewrite Enabled
  5. CURL Module Active
  6. allow_url_fopen Enabled
  7. file_get_contents Enabled
  8. Php Exec Enabled

4) Installation - top

In order to install the CMS, please follow these steps:

  1. Unzip the file you download from Envato.
  2. Search for the folder cms in the main script folder that you have downloaded.
  3. Now upload the cms folder into your server using FTP.
  4. Alternately you can zip the cms folder and upload it to server via cpanel. Then extract this and use.
  5. You can use our cms into root domain or subdomain.
  6. Become sure that you have uploaded the .htaccess file correctly. When you will extract the zip file staying on server via cpanel, the .htaccess file may not be extracted properly in the server and then the script will show error. So after extracting, you may have to manually upload the .htaccess file into the server that comes with the script file.
  7. Create mysql database using cpanel or your hosting provider's system. Then import the servixo_bizplus.sql file (that you got into the script's database folder) into your created database.
  8. Now you will have to setup the config and database file. If you are familier with codeigniter, you should know about it. But don't worry! We will tell step by step.
  9. Go to the cms folder, then application and then config folder.

6) Email Setting - top

You must have to setup the email correctly. Otherwise the system will not work. A SMTP email system is added here. You can activate or deactivate the SMTP from the email setting page. Go to this page: Website Settings and then Email tab. Then change your information.

7) Payment Setting - top

You must have to setup the payment API information correctly. Otherwise customers will not be able to make an order for the shop section. Go to this page: Website Settings and then Payment tab. Then edit the paypal and stripe information in there.
